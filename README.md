# MeDVeD

Лёгкий Windows-клиент для прокси-сервисов на базе [sing-box 1.13](https://github.com/SagerNet/sing-box).
Ставится за пару кликов, всё внутри, ничего не нужно докачивать.

## Возможности

- **Протоколы:** VLESS · VMess · Trojan · Shadowsocks · Hysteria2 · TUIC
- **Подписки:** простой текст, base64 и Clash YAML, авто-обновление по расписанию
- **Авто-режим (URLTest)** — выбирает самый быстрый сервер из списка
- **Авто-фильтр Европы** — новые подписки сразу режут «не Европа» и «Обход»
- **Маршрутизация:** правила по доменам / IP / портам / именам процессов, обход RU-сайтов через geosite-ru
- **Kill switch** — блокирует интернет при неожиданном падении VPN
- **Self-heal** — одна попытка автоматического переподключения при крахе
- **Авто-обновление** через GitHub Releases (кнопка + красный кружок на меню)
- **Светлая/тёмная тема, ru/en интерфейс**
- **Автозапуск с Windows + старт свёрнутым в трей**
- **Tray-меню с быстрым выбором сервера** (ПКМ по иконке)
- **Внешний IP после подключения** — мгновенный сигнал «VPN реально работает»
- **TLS-handshake пинг** (то же что показывает Karing — реалистичные 50–200 мс)
- **Параллельная проверка пингов** + усреднение, чтобы не верить случайным выбросам

## Установка

1. Скачать `MeDVeD-Setup-X.Y.Z.exe` со [страницы релизов](https://github.com/Jellytyt/MeDVeD/releases/latest)
2. Запустить установщик. При первом запуске Windows SmartScreen может показать
   «Неизвестный издатель» — нажми **«Подробнее → Выполнить в любом случае»**.
   Это штатное предупреждение для любого неподписанного приложения, не вирус.
3. Установщик поставит MeDVeD в `%LOCALAPPDATA%\Programs\MeDVeD` (без запроса прав
   администратора) и создаст ярлык в меню «Пуск».

> Сам VPN (TUN-режим) требует прав администратора — приложение запросит UAC при
> подключении. Это отдельно от установки: ставится оно без прав, а права нужны
> только на поднятие туннеля.

Личные данные хранятся отдельно от программы, в `%LOCALAPPDATA%\vless_manager\`:

- `profiles.json` — твои импортированные сервера
- `settings.json` — параметры приложения
- `app.log` — журнал с ротацией по 1 МБ

При обновлении через UI приложение скачивает свежий установщик и тихо обновляется
(закрывает старую версию, заменяет файлы, перезапускается) — твои настройки
остаются на месте.

## Что внутри

- Python 3.13 / Tkinter + [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
- `sing-box.exe` (текущая стабильная версия от SagerNet) лежит в папке программы
  рядом с `MeDVeD.exe` (в `_internal\`) — при запуске ничего никуда не распаковывается
- [pystray](https://github.com/moses-palmer/pystray) для иконки в трее, [PyYAML](https://pyyaml.org/) для Clash подписок

Приложение ставится **папкой** (onedir), а не одним самораспаковывающимся exe —
поэтому при запуске нет временной распаковки в `_MEI...` и связанной с ней ошибки
первого запуска `Failed to load Python DLL`. Размер установщика — несколько
десятков МБ (зависит от версии sing-box).

## Поддержка

Багрепорты и предложения — в [Issues](https://github.com/Jellytyt/MeDVeD/issues).
